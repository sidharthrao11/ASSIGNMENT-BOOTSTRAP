Q.1 What are the advantages of Bootstrap?

Ans:- Bootstrap, a popular front-end framework, offers several advantages:-

1.Rapid Development: Bootstrap provides pre-styled components and a grid system, allowing developers to quickly
build responsive and visually appealing websites without spending time on writing CSS from scratch.

2.Responsiveness: Bootstrap's grid system ensures that websites built with it are responsive and adapt well to different
screen sizes and devices, reducing the need for separate designs for desktop and mobile.

3.Consistency: Bootstrap's extensive library of components ensures consistency in design across the entire website.
This consistency leads to a better user experience as users navigate through different pages.

4.Customization: While Bootstrap comes with a default set of styles and components, it's highly customizable.
Developers can easily override the default styles to match the specific design requirements of their project.

5.Cross-browser Compatibility: Bootstrap is designed to be compatible with all major browsers, ensuring a consistent
experience for users regardless of the browser they use.
